void ElevationData2D::ReadFile::getData ( Options &  opt)

reads a tiff file and writes it to an array. With a longitude less than 50 and greater than -51 all values are copied into the array. For all other images, a linear interpolation over the neighboring pixels is performed and the mean value is formed. This is stored twice in the array.
